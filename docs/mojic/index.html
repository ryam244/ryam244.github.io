❓ よくある質問
難易度の違いを教えてください
かんたん（4×4グリッド）/ ふつう（5×4グリッド）/ むずかしい（5×5グリッド＋シャッフル）の3段階です。むずかしいではパネルがシャッフルされるため反射神経がより試されます。
ウィークリーランキングはいつリセットされますか？
毎週月曜日にリセットされます。週替わりのゲームモードで世界中のプレイヤーと競い合い、上位プレイヤーは殿堂入りに記録されます。
プレミアムを購入後に端末を変えた場合は？
同じApple ID / Google アカウントでアプリを再インストール後、設定画面の「購入を復元」をタップすることでプレミアムが引き継がれます。
オフラインでも遊べますか？
基本的なゲームプレイはオフラインでも楽しめます。ランキングやデイリーチャレンジへの参加にはインターネット接続が必要です。
ログインボーナスはどうやって受け取りますか？
毎日アプリを起動するとログインボーナス画面が表示されます。7日サイクルで報酬が変わり、連続ログインでより多くのボーナスが獲得できます。
