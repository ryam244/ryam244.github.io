❓ よくある質問
ウィジェットをホーム画面に追加する方法は？
iOSのホーム画面を長押しして編集モードに入り、「＋」ボタンからMojicを選択してください。作成したパネルを追加できます。
作成できるウィジェットの数に制限はありますか？
無料版では3つまで作成できます。それ以上作成したい場合はプレミアム版をご利用ください。
ウィジェットの内容は自動で更新されますか？
テキストは手動で更新します。アプリを開いて内容を変更すると、ホーム画面のウィジェットに反映されます。
作成したウィジェットはバックアップできますか？
設定画面からデータのエクスポートが可能です。機種変更時にも作成したデザインを引き継げます。
