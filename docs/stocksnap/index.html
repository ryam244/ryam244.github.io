❓ よくある質問
データのバックアップはできますか？
はい、設定画面の「データバックアップ」からエクスポート・インポートが可能です。機種変更時やアプリの再インストール時にご活用ください。
数量を0.5個など小数で入力できますか？
はい、対応しています。数量をタップして直接入力するか、ボタンで0.5刻みで調整できます。「100g」や「0.5本」などの管理も可能です。
単位を追加・カスタマイズしたいのですが？
設定画面の「単位リスト」から独自の単位を追加できます。追加した単位はすべての画面で利用可能です。
通知が届きません
iPhoneの「設定」→「StockSnap」→「通知」で通知が許可されているか確認してください。また、アプリ内の設定でリマインダーがオンになっているかもご確認ください。通知は毎朝設定した時刻に送信されます。
食材のカテゴリを増やすことはできますか？
現在は「冷蔵」「冷凍」「常温」の3カテゴリに対応しています。カテゴリのカスタマイズは今後のアップデートで対応予定です。
ダークモードに対応していますか？
はい、iPhoneの設定に連動して自動的にダークモードに切り替わります。手動での切り替えはアプリ内設定からも可能です。
データはクラウドに保存されますか？
いいえ、すべてのデータはお使いのデバイス内にのみ保存されます。クラウドへの自動同期は行っておりません。機種変更の際はエクスポート機能でデータを移行してください。
