❓ よくある質問
アプリを閉じてもタイマーは動きますか？
はい、バックグラウンドでタイマーは動作し続け、終了時に通知でお知らせします。iPhoneの通知設定がオンになっている必要があります。
複数のタイマーを同時に起動できますか？
はい、複数のタイマーを同時に動かすことができます。それぞれ個別に通知が届きます。
インターバルタイマーはどうやって設定しますか？
タイマー作成画面で「繰り返し」をオンにして、インターバル回数と休憩時間を設定してください。ポモドーロ法などにも活用できます。
通知が届きません
iPhoneの「設定」→「Kakeru」→「通知」で通知が許可されているか確認してください。また、おやすみモードや集中モードがオンの場合は通知が届かないことがあります。
