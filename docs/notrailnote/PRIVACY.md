# プライバシーポリシー

**最終更新日: 2026年2月9日**

## 1. はじめに

Notrailnote（以下「当社」）は、モバイルアプリケーション「NotrailNote」（以下「本アプリ」）におけるユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明するものです。

## 2. 収集する情報

本アプリは、以下の情報を収集・保存する場合があります。

### 端末内に保存される情報

- 議事録・ノートの内容（タイトル、本文、タグ）
- ノートの作成日時・更新日時
- アプリの設定情報（テーマ、フォントサイズ等）

### GitHub連携時に使用される情報

- GitHubパーソナルアクセストークン（端末内に暗号化して保存）
- GitHubユーザー名
- リポジトリ情報

## 3. 情報の保存場所

本アプリで取り扱う情報は、以下の場所に保存されます。

### 端末内

- すべてのノートデータは、ユーザーの端末内にのみ保存されます
- GitHub認証情報は、端末のセキュアストレージに暗号化して保存されます

### GitHub（ユーザーの選択による）

- GitHub連携機能を有効にした場合、ノートはユーザー自身のGitHubリポジトリに保存されます
- これはユーザーの任意の操作によるものであり、当社のサーバーを経由しません

## 4. 情報の利用目的

収集した情報は、以下の目的でのみ使用されます。

- ノートの保存・編集・表示機能の提供
- GitHub連携によるバックアップ機能の提供
- アプリ設定の保持
- ユーザー体験の向上

## 5. 第三者への情報提供

当社は、以下の場合を除き、ユーザーの情報を第三者に提供することはありません。

- ユーザーの同意がある場合
- 法令に基づく開示請求があった場合

なお、本アプリはユーザーのノート内容を当社のサーバーに送信しないため、当社がノート内容にアクセスすることはありません。

## 6. セキュリティ

本アプリは、ユーザーの情報を保護するために以下の対策を講じています。

- GitHub認証情報の暗号化保存
- 生体認証によるアプリロック機能
- 端末内でのオフライン処理

## 7. お子様のプライバシー

本アプリは、特にお子様向けに設計されたものではありませんが、お子様でもご利用いただけます。お子様が本アプリを利用する場合は、保護者の方の監督のもとでご利用ください。

## 8. 外部サービスとの連携

本アプリは、以下の外部サービスと連携する機能を提供しています。

### GitHub

- GitHub連携機能を利用する場合、GitHubのプライバシーポリシーも適用されます
- GitHub: https://docs.github.com/ja/site-policy/privacy-policies/github-privacy-statement

## 9. データの削除

ユーザーは、以下の方法でデータを削除することができます。

- 個別のノート: アプリ内から削除
- GitHub連携データ: 設定画面からGitHub連携を解除
- すべてのデータ: アプリをアンインストール

## 10. プライバシーポリシーの変更

当社は、必要に応じて本プライバシーポリシーを変更することがあります。重要な変更がある場合は、アプリ内でお知らせいたします。
