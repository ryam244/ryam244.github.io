# NotrailNote

**議事録が資産になる**

## アプリ概要

NotrailNoteは、「書いて終わり」になりがちな議事録を、チームの資産として活用できるミーティングノート特化アプリです。

### コンセプト

**「振り返れる議事録」**

既存の議事録ツールの課題を解決します：
- 書いて終わり、振り返らない
- 決定事項が埋もれる
- 「あの時何決めたっけ？」が頻発
- アクションアイテムの追跡が困難

## 主な機能

### ローカルファースト
- すべてのデータは端末内に保存
- オフラインでも完全に動作
- プライバシーを最優先

### Markdownエディタ
- シンプルで使いやすいエディタ
- リアルタイムプレビュー
- 議事録に最適化されたテンプレート

### GitHubバックアップ
- 任意でGitHubと連携
- 自動バックアップ
- 変更履歴の完全な追跡
- チームでの共有も可能

### セキュリティ
- 生体認証によるアプリロック
- 認証情報の暗号化保存
- 当社サーバーへのデータ送信なし

## ターゲットユーザー

- スタートアップのPM/EM
- 少人数チームのリーダー
- フリーランスのコンサルタント
- リモートワーク中心のチーム

## 技術情報

- **プラットフォーム**: iOS（App Store）
- **開発フレームワーク**: Flutter
- **データ保存**: SQLite（端末内）
- **バックアップ**: GitHub API
