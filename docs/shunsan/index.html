❓ よくある質問
計算履歴はどこで確認できますか？
メイン画面の履歴ボタンから過去の計算一覧を確認できます。計算結果をタップすると再利用できます。
計算履歴はいくつまで保存されますか？
最大100件まで保存されます。古いものから自動的に削除されます。履歴を手動で削除することも可能です。
小数点以下の桁数は変更できますか？
設定画面から表示する小数点以下の桁数（0〜10桁）を変更できます。
データはクラウドに保存されますか？
いいえ、すべてのデータはデバイス内にのみ保存されます。
