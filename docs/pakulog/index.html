❓ よくある質問
何匹まで登録できますか？
複数のペットを登録して管理できます。ペットを切り替えながら個別に食事を記録することができます。
食事のリマインダーを設定できますか？
はい、朝・昼・夜など複数の給餌時刻を設定できます。iPhoneの通知設定がオンになっている必要があります。
フードの種類をカスタムで追加できますか？
はい、フード名・メーカー・カロリー情報などをカスタムで登録できます。登録したフードは記録時にかんたんに選択できます。
記録データを獣医師に共有できますか？
エクスポート機能でデータをPDF・CSV形式で書き出し、メールなどで共有できます。定期健診の際にご活用ください。
データのバックアップはできますか？
設定画面からデータのエクスポートが可能です。機種変更時はエクスポートしたデータをインポートして引き継いでください。
