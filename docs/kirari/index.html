❓ よくある質問
無料でどこまで使えますか？
無料プランでは体重の記録とAIコメントの基本機能をご利用いただけます。より詳細な分析や広告なし利用はライト・プロプランをご検討ください。
AIのコメントはどのように生成されますか？
記録した体重・気分・メモをもとに、AIが励ましや気づきのコメントを自動生成します。個人情報はAIの学習には利用されません。
データはどこに保存されますか？
記録データはデバイス内に保存されます。AIコメント生成のため記録内容をサーバーへ送信しますが、個人を特定できる情報は含まれません。
プランはいつでも変更できますか？
はい、App Storeのサブスクリプション設定からいつでも変更・解約できます。解約後は当該期間の終了まで機能をご利用いただけます。
体重のグラフはどこで見られますか？
ホーム画面に週間グラフが表示されます。ライト・プロプランでは月次・年間グラフも確認できます。
